# 🎯 配置概览

配置就是告诉机器人"你是谁"、"怎么说话"、"用什么AI大脑"。就像给新来的朋友介绍自己一样简单！

## 📋 配置文件清单

MaiBot 有两个主要的配置文件，都放在 `config/` 文件夹里：

| 📁 文件 | 📝 作用 | 🔗 详细说明 |
|---------|---------|------------|
| `bot_config.toml` | 机器人的基本信息、性格、聊天习惯 | [查看详情](./bot-config.md) |
| `model_config.toml` | AI模型设置（哪个AI、怎么用） | [查看详情](./model-config.md) |

::: tip 💡 小贴士
第一次启动时，如果找不到配置文件，MaiBot 会自动生成默认配置，不用担心！
:::

## 🧩 bot_config.toml 里面有什么？

这个文件控制机器人的"人格"，包含这些部分：

| 🏷️ 配置段 | 🎯 控制什么 | 💡 简单理解 |
|-----------|------------|------------|
| `[bot]` | 基本信息 | "我叫什么名字，QQ号是多少" |
| `[personality]` | 性格设定 | "我是活泼还是文静？说话什么风格？" |
| `[chat]` | 聊天行为 | "我话多不多？什么情况下回复？" |
| `[visual]` | 看图能力 | "能不能看懂图片？怎么看？" |
| `[memory]` | 记忆系统 | "要不要记住聊天内容？记多少？" |
| `[message_receive]` | 消息过滤 | "哪些消息我不理？" |

## ⚡ 热重载 - 改完立即生效！

MaiBot 支持**热重载**，意思是：
- ✅ 修改配置文件后，**不用重启**机器人
- ✅ 保存文件后，**自动重新加载**新配置
- ✅ **立即生效**，马上看到效果

::: warning ⚠️ 注意
有些配置（比如换AI模型）可能需要重新初始化相关服务才能完全生效。
:::

## 🌐 WebUI 图形界面

如果你不喜欢手动改文件，MaiBot 还提供了网页版的配置界面：

- 🌐 默认地址：`http://127.0.0.1:8001`
- 🖱️ 点点鼠标就能改配置
- 📱 手机、电脑都能用

::: warning 🔒 安全提示
在公网使用时，建议用 Nginx 等反向代理，不要直接暴露 8001 端口！
:::

## 🔄 配置文件版本管理

配置文件里有个 `[inner]` 段，里面的 `version` 字段表示文件版本。

当 MaiBot 升级时：
1. 🔍 自动检测版本差异
2. 💾 备份旧配置到 `config/old/` 文件夹
3. 🆕 生成新的配置文件
4. 🚪 程序退出，需要手动重启

这样既能享受新功能，又不会丢失原来的配置！
