# 配置概览
## 📋 配置文件清单

MaiBot 有两个主要的配置文件，都放在 `config/` 文件夹里：


| 📁 文件 | 📝 作用 | 🔗 详细说明 |
|---------|---------|------------|
| `bot_config.toml` | 机器人的基本信息、性格、聊天、记忆、学习..... | [查看详情](./bot-config.md) |
| `model_config.toml` | AI模型设置，设置麦麦使用的LLM | [查看详情](./model-config.md) |

::: tip 💡 小贴士
这些配置文件需要启动一次MaiBot之后才会生成。如果无法找到，请先启动一次。
:::

::: tip 💡 麦麦现在支持配置热重载
麦麦现在支持配置热重载。修改配置文件后无需重启，保存即可自动重新加载新配置。
(有些配置（比如换AI模型）可能需要重新初始化相关服务才能完全生效。)
:::


## 🌐 WebUI 图形界面

如果你不喜欢手动改文件，MaiBot 还提供了网页版的配置界面（1.0.0预发布版的webui尚未完成）：

- 🌐 默认地址：`http://127.0.0.1:8001`
- 🖱️ 点点鼠标就能改配置
- 📱 手机、电脑都能用

::: warning 🔒 安全提示
在公网使用时，建议用 Nginx 等反向代理，不要直接暴露 8001 端口！
:::
