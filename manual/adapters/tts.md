# 🔊 语音合成

让 MaiBot 能发语音消息，开口说话！

## 什么是语音合成？

语音合成（TTS）就是把文字变成声音：
- 你输入："你好，我是 MaiBot"
- 机器人发出：🔊"你好，我是 MaiBot"

就像给机器人装了个"嘴巴"！

## 能做什么？

### 🎤 文字转语音
- 把回复的文字读出来
- 支持多种声音
- 可调节语速语调

### 🎧 语音转文字（可选）
- 听懂用户的语音消息
- 把语音转换成文字处理
- 支持多种语言

## 支持的语音服务

### 🏠 GPT-SoVITS（推荐）
**自己搭建，完全免费**

特点：
- ✅ 完全开源免费
- ✅ 可以克隆任何人的声音
- ✅ 中文效果特别好
- ⚠️ 需要电脑有显卡（4G 显存以上）

**简单设置**：
1. 下载 GPT-SoVITS
2. 启动服务（默认端口 9880）
3. 在配置里填上地址：`http://127.0.0.1:9880`

### ☁️ 豆包 TTS
**云端服务，按量付费**

特点：
- ✅ 效果自然，像真人说话
- ✅ 不用自己搭建
- ✅ 响应速度快
- 💰 要花钱（但很便宜）

**申请使用**：
1. 注册火山引擎账号
2. 开通语音合成服务
3. 获取 API Key，填到配置里

### ☁️ 千问 Omni
**阿里云出品，质量上乘**

特点：
- ✅ 大厂出品，稳定可靠
- ✅ 中文语音很自然
- ✅ API 简单易用
- 💰 按用量收费

**开通服务**：
1. 注册阿里云账号
2. 开通 DashScope 服务
3. 创建 API Key 并配置

## 怎么设置？

### 第一步：选择语音服务
根据你的需求选择：
- 有显卡 → GPT-SoVITS（免费）
- 没显卡 → 豆包或千问（付费）

### 第二步：配置语音服务
在 `model_config.toml` 里添加：

```toml
# 以 GPT-SoVITS 为例
[[api_providers]]
name = "我的语音服务"
base_url = "http://127.0.0.1:9880"
api_key = ""  # 本地服务不用填

[[models]]
name = "语音合成"
api_provider = "我的语音服务"
model_identifier = "tts"
```

### 第三步：开启语音功能
在 `bot_config.toml` 里设置：

```toml
[voice]
enable_asr = true  # 如果想让机器人听懂语音，设为 true
```

### 第四步：重启 MaiBot
保存配置，重启机器人，就能说话了！

## 使用技巧

### 声音选择
不同服务提供不同声音：
- 男声/女声
- 年轻人/老年人
- 温柔/活泼
- 各种方言

### 语速调节
可以调节：
- 说话速度（快慢）
- 音调高低
- 音量大小

### 场景应用
- **群聊活跃气氛** - 偶尔发语音增加趣味
- **私聊更亲切** - 语音回复更有人情味
- **朗读功能** - 读诗、读故事、读新闻

## 注意事项

⚠️ **重要提醒**：
- 语音消息比较大，会占用更多流量
- 频繁发语音可能影响聊天体验
- 注意平台对语音时长的限制

💡 **省钱技巧**：
- GPT-SoVITS 本地部署完全免费
- 云服务选择便宜套餐
- 合理控制语音使用频率

## 常见问题

**Q: 语音不工作怎么办？**
A: 检查服务是否启动，地址是否正确，API Key 是否有效

**Q: 声音不好听能换吗？**
A: 可以，大多数服务都提供多种声音选择

**Q: 语音转文字准确吗？**
A: 普通话很准，方言可能差点，背景音乐会影响识别

**Q: 会泄露隐私吗？**
A: 本地服务完全私密，云服务注意查看隐私政策
